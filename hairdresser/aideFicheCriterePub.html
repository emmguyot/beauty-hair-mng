Aide de la fiche publipostage

Cette fiche permet d'extraire des clients pour effectuer des publipostages. Ce publipostage s'appuie sur les fonctionnalités offertes par des logiciels comme Microsoft Word, Sun StarOffice ou OpenOffice. Ainsi vous devez posséder un de ces logiciels pour réaliser votre publipostage. Ceci vous permet de créer un mailing papier ou email aussi personnalisé que possible.

  • Pour réaliser un publipostage avec Microsoft Word :
    • Saisir les informations représentant les critères de sélection des clients
    • Cliquer sur . Une fenêtre s'affiche pour vous demander ce que vous souhaitez faire du fichier extrait.
    • Cliquer sur Enregistrer
    • Indiquer le répertoire où le fichier sera stocké
    • Fermer la fenêtre
    • Ouvrir Word et charger la lettre à envoyer aux clients. Pour vous aider à démarrer avec cette fonction, voici un exemple de document
    • Sélectionner l'icône "Ouvrir la source de données" et indiquer le fichier extrait ci dessus
    • Sélectionner l'icône "Fusionner vers un nouveau document" et indiquer que vous souhaitez tous les enregistrements
    • Un nouveau document est créé avec autant de pages que de clients. Après un contrôle visuel du rendu, vous pouvez imprimer ce document normalement.
  • Pour réaliser un publipostage avec Sun StarOffice :
    • Saisir les informations représentant les critères de sélection des clients
    • Cliquer sur . Une fenêtre s'affiche pour vous demander ce que vous souhaitez faire du fichier extrait.
    • Cliquer sur Enregistrer
    • Indiquer le répertoire où le fichier sera stocké
    • Fermer la fenêtre
    • Ouvrir Star Office et charger la lettre à envoyer aux clients. Pour vous aider à démarrer avec cette fonction, voici un exemple de document
    • Lors de la première utilisation du publipostage avec StarOffice :
      • Sélectionner Fichier / Nouveau / Base de données. Une fenêtre s'ouvre
      • Dans cette fenêtre, cliquer sur l'onglet Type et choisir Texte
      • Dans ce même onglet, cliquer sur Parcourir afin de sélectionner le répertoire où vous avez enregistré le fichier clients.csv
      • Toujours dans cette fenêtre, cliquer sur l'onglet Texte et choisir csv dans la liste déroulante Extension
      • Dans ce même onglet, choisir le caractère ";" comme séparateur de champ
      • Valider la fenêtre par Ok. La base de données apparait maintenant dans l'Explorer à gauche de la fenêtre avec le nom <Base de données>
      • Afficher la lettre chargée préalablement en cliquant sur le nom de la lettre en bas de l'écran
    • Appuyer sur la touche F4, le volet des données s'affiche au dessus de la lettre
    • Dans l'explorer à gauche, cliquer sur le + devant <Base de Données>, puis cliquer sur le + devant Tables
    • Sélectionner clients sous Tables. Ceci affiche vos données dans la partie au dessus de la lettre
    • Dans cette partie Données, cliquer sur l'icône Mailing composée de 3 enveloppes empilées
    • Sur la fenêtre qui s'est affichée, cliquer sur Tous et valider par Ok
    • Imprimer l'ensemble des lettres comme d'habitude
