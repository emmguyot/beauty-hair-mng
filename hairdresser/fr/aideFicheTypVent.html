Aide de la fiche type de prestations

Cette fiche permet de créer, de modifier ou de supprimer un type de prestations.

  • Pour modifier un type de prestations :
    • Saisir ou modifier les informations
    • Les civilités associées permettent de n'afficher des types de prestations que pour les clients d'une certaine civilité.
    • Cliquer sur et vérifier que l'enregistrement s'est bien effectué grâce au message sous le titre
  • Pour supprimer un type de prestations :
    • Vérifier que ce type de prestations n'a jamais été utilisé
    • Cliquer sur et vérifier que la suppression s'est bien déroulée grâce au message sous le titre

Voir aussi les trucs et astuces.
