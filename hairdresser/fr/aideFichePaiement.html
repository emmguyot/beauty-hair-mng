Aide de la fiche Paiement

Cette fiche présente les informations de paiement d'une ou de plusieurs factures : La partie haute indique les informations de paiement. La partie principale affiche une liste de factures en attente de paiement.

  • Pour régler plusieurs factures par un paiement unique :
    • Saisir ou modifier le mode de paiement
    • Cocher les factures concernées
    • Cliquer sur et vérifier que l'enregistrement s'est bien effectué grâce au message sous le titre.
  • Pour imprimer la facturette :
    • Cliquer sur quand toutes les informations sont bonnes et que le paiement est saisi. Ceci enregistre les modifications au passage.
  • Pour afficher la fiche client d'une facture de la liste :
    • Cliquer sur le nom qui est affiché en bleu.
  • Pour afficher une facture listée :
    • Cliquer sur la date de la facture qui est affichée en bleu.
  • Pour imprimer le chèque de votre client :
    • Cliquer sur "imprimer chèque" à coté du total. Ceci n'est possible que si le paiement a été enregistré.
  • Pour changer le mode de paiement après enregistrement :
    • Modifier le mode de paiement en conséquence
    • Cliquer sur et vérifier que l'enregistrement s'est bien effectué grâce au message sous le titre. Les soldes de caisse et les stocks sont modifiés afin d'annuler les mouvements effectués lors du premier paiement.
  • Pour supprimer un paiement :
    • Cliquer sur et vérifier que la suppression s'est bien effectuée grâce au message sous le titre. Les soldes de caisse et les stocks sont modifiés afin d'annuler les mouvements effectués lors du premier paiement. Les factures correspondantes reviennent dans l'en cours.
