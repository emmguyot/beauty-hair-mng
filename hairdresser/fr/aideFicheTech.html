Aide de la fiche technique

Cette fiche présente les informations techniques que vous avez saisies au niveau des prestations afin que vous l'imprimiez. Il affiche les informations techniques par catégorie de prestation (Couleur, Permanente, ...).

  • Pour afficher plus d'informations :
    • Modifier le nombre d'historiques différents qui est par défaut à 2.
  • Pour afficher moins d'informations :
    • Décocher les catégories de prestations ne concernant pas le rendez-vous du jour
  • Pour imprimer la fiche historique :
    • Cliquer sur Le format de l'impression est le même que la facturette : format A6.
