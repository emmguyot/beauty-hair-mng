Aide de la liste des paramètres de l'application

Cet écran affiche la liste des paramètres de l'application. Les paramètres sont les suivants :

  • Affichage prix en bas de facture : La valeur O (pour Oui) indique que le prix doit être affiché sur l'écran fiche facture.
  • Message ticket : Indique le message s'affichant en bas du ticket de caisse.
  • Mot de passe Opérations Exceptionnelles : Permet de modifier le mot de passe
  • Répartition de la remise (P ou F) : La valeur P indique que la remise est répartie proportionnellement sur chaque ligne de la facture. La valeur F indique que la remise est attribuée au type de prestation de la facture.
  • Connexion automatique : La valeur O indique que le mot de passe n'est plus à saisir pour utiliser l'application
  • Disque amovible (ex. : A:/) : Indique la lettre du lecteur (et éventuellement le répertoire) ou les sauvegardes sur disque amovible seront faites. Ceci peut correspondre à un lecteur de disquette, un disque réseau ou encore une clé USB.
  • Largeur d'impression des fiches (cm) : Indique la largeur des tickets de caisse imprimés.
  • Message taxes comprises du ticket : Indique le message affiché pour indiqué que les taxes et le service sont compris dans le montant du ticket.
  • URL des messages d'info : Adresse du site fournissant les différentes informations sur le logiciel.
  • URL des mises à jour du logiciel : Adresse du site fournissant les différentes versions du logiciel permettant une mise à jour transparente du logiciel.

Cet écran vous permet de :

  • Consulter ou modifier la valeur d'un paramètre :
    • Cliquer sur le nom du paramètre pour afficher la fiche du paramètre
