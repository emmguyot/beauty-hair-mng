Help for the form

The form presents the whole of information in order to allow you to act on this information: Modify, Create, Remove, Duplicate, Print...

The obligatory data are posted in stand-up letters and the optional data are in italics. Thus, for example to create a customer, only the name is obligatory. However it is highly advised to obtain the most information possible in order to be able to carry out mailings, statistics...

On certain screens containing a list of modifiable data, a green line is posted. This working line allows the input of a new line which is then added with the button at the end of the line. Without validation on this button, information of the new line is likely to be lost.

In bottom of the screen, the possible actions vary from one screen to another. The actions are as follows:

  • Update: After the modification of information, this action carries out the recording of the modifications of the card in progress.
  • Duplicate: In order to accelerate the successive inputs, this action creates a copy of the card in progress and records it. You can then modify it without modifying the initial card.
  • Print: Launch the print of information of the card in a preset format. This action is available only when one format exists: The invoice for example. However, you can always print by clicking on the right button of the mouse and by using the Print option which is displayed in the menu.
  • Remove: This action removes the card in progress. So that the deletion is allowed, the card should not be used by other information. For example you cannot remove a customer if it has invoices.
  • List: Post the list of information
