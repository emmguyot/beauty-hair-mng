Help of the form

The form presents the whole of information in order to allow to act on this information: To modify, Create, Remove, Duplicate, Print...

The obligatory data are posted in right letters and the optional data are in italics. Thus, for example to create a customer, only the name is obligatory. However it is highly advised to obtain the most information possible in order to be able to carry out mailings, statistics...

On certain screens containing a list of modifiable data, a green line is posted. This line of work allows the seizure of a new line which is then added with the button at the end of the line. Without validation on this button, information of the new line is likely to be lost.

In bottom of the screen, the possible actions vary from one screen to another. The actions are as follows:

  • Update: After the modification of information, this action carries out the recording of the modifications of the card in progress.
  • Duplicate: In order to accelerate the successive seizures, this action creates a copy of the card in progress and records it. You can then modify it without modifying the initial card.
  • Print: Launch the impression of information of the card in a preset format. This action is available only when one format exists: The invoice for example. However, you can always print while clicking on the right button of the mouse and by using the Imprimer option which is posted then.
  • Remove: This action removes the card in progress. So that the suppression is possible, the card should not be used by other information. For example you cannot remove a customer if it has invoices.
  • List: Post the list of information
