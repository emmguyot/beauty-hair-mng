Help for the mailing form

This form lets you extract customers in order to do mailings. This mailing underlies on the features provided by software such as Microsoft Word, Sun StarOffice or OpenOffice. So you have to own one of these to do mailing. They let you create post mailing or email mailing that are personnalized.

  • To create a mailing with Microsoft Word :
    • Enter the critereas to select customers
    • Click on . A window opens to ask what to do with the extracted file.
    • Click on Save
    • Select the directory where the file will be stored
    • Close the window
    • Open Word and load a letter to send to these customers. To help you in this task, Here is a sample
    • Select the icon "Open data source" and select the file you've just extracted
    • Select the icon "Merge to a new document" and notify that you want every records
    • A new document is produced with as many pages as customers the file includes. After a visual check of the global document, you may print the document as usual.
  • To create a mailing with Sun StarOffice :
    • Enter the critereas to select customers
    • Click on . A window opens to ask what to do with the extracted file.
    • Click on Save
    • Select the directory where the file will be stored
    • Close the window
    • Open Star Office and load a letter to send to these customers. To help you in this task, here is a sample
    • For the first time you use the mailing with StarOffice :
      • Select File / New / Database. A window opens
      • In this window, click on the tab Type and choose Text
      • In the same tab, click on Browse to select the folder where you've saved the file clients.csv
      • Still in this window, click the tab Text and choose csv in the combo box Extension
      • Choose the character ";" as the field separator
      • Valid the window with Ok. The database now appears in the Explorer on the left of the windows with the name <Database>
      • Display the letter you've loaded from the bottom tool bar
    • Press the F4 key, the data part shows up above the letter
    • In the explorer on left, click on the + in front of <Database>, then click on the + in front of Tables
    • Select clients in Tables. Your data are displayed above the letter
    • In this Data part, click on the icon Mailing that is build with 3 stacked wraps
    • In the window that pops up, click on All and confirm with Ok
    • You may print the document as usual.

You can check the hints and tips too.
