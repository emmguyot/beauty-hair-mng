Help for the list of parameters of the application

This list displays the parameters of the application. The available parameters are the following :

  • Display the price at the bottom of the invoice : The value O (for Yes in french) states that the price have to be displayed on the invoice form.
  • Message ticket : Defines the message at the end of the bill.
  • Password for special operations : Defines the password for special operations
  • Spread over of the discount (P=Proportionnal or F=Fixed) : The value P states that the discount is splitted equally on each line of the invoice. The value F states that the discount is linked to the kind of service of the invoice.
  • Auto connect : The value O (for Yes in French) states that no password is required to use the application
  • Removable media (eg: A:/) : Defines the drive letter (and eventually the folder) where backups on Removable disks should be made. It may be a floppy disk, a network drive or a USB key.
  • Print width of the bills (cm) : Defines the width of the paper you have to print the bills.
  • Message tax included on the bill : Defines the message printed on the bill to states that taxes and service are included in the price.
  • URL for news : Address of the site that provides news about the software.
  • URL for software updates : Address of the site that provides various release of the software in order to easily update the software.
