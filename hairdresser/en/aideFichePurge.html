Help for the data remover

This form lets you remove old data in order to work and to manage backups faster. Before any data removal, you have to make a backup in order to keep data as long as the law ask for.

  • To start the data remover :
    • Enter the date to witch data are not useful anymore and may be removed.
    • Check the data you want to remove.
    • Click on , and, after a while, verify that all is correct by reading the message and the number of record that has been removed

    N.B. : NEVER stop a data removal, this may stop the application from working any more.

If a huge amount of data will be removed, this operation will require time. You may proceed by part if you select only the a part of data or if you change the date.

You can check the hints and tips too.
